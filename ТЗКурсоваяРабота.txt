Кроссплатформенная автоматизированная информационная система “Выпускники «МЦК-КТИТС»”.

Концептуальная задача.
Разработать АИС “Выпускники «МЦК-КТИТС»” Данная информационная система предназначена для людей, которые хотели бы узнать больше о выпускниках техникума, найти информацию о своих родственниках. Приложение позволяет выводить список всех выпускников техникума, с возможностью поиска по ФИО, группе, дате выпуска. Реализована ролевая система входа в приложение. У клиента нет возможности редактировать и добавлять новых выпускников, но есть возможность отправить запрос на добавление или редактирование выбранного выпускника. Администратору доступны все функции клиента, а также редактирование, добавление и удаление определенного выпускника. Реализована система для вывода одногруппников определенного выпускника.

Общие:
1.	Цветовая палитра белый/синий/красный.
2.	Возможность регистрации как клиент и администратор.
3.	Реализация списка всех выпускников
4.	Реализация добавления выпускников в базу данных:
4.1 Возможность клиента отправить запрос на добавление выпускника, при помощи приложения.
4.2. Возможность добавления администратором выпускника при помощи приложения. 
5.	Возможность осуществить поиск по ФИО, группе, году выпуска и году поступления.
6.	Реализация возможности поиска одногруппников по выбранному выпускнику

Начальный экран/Экран загрузки.
1.	При старте приложения появляется логотип техникума и название приложения
Форма регистрации:
1.	Текстовые поля:
1.1 Для ввода имени.
1.2 Для ввода номера телефона
1.3 Для ввода E-Mail адреса
2.	Кнопки:
2.1 Для очистки содержимого
2.2 Для подтверждения регистрации

Домашняя страница (Для клиента).
1.	Кнопка автоматического поиска одногруппников выбранного выпускника (открывает форму со списком всех одногруппников)
2.	Кнопка поиска по ФИО, группе, дате выпуска (открывает форму для поиска)
3.	Информация о выпускнике(поля):
1.1 Фамилия, имя, отчество
1.2 Дата выпуска
1.3 Группа
1.4. Достижения
1.5 Фотография выпускника
4.	Нижняя панель:
4.1 Кнопки перелистывания выпускников



Домашняя страница (Администратор).

1.	Кнопка автоматического поиска одногруппников выбранного выпускника (открывает форму со списком всех одногруппников)
2.	Кнопка поиска по ФИО, группе, дате выпуска (открывает форму для поиска)
3.	Кнопка добавления нового выпускника (открывает новую страницу с полями ФИО, дата выпуска, группа, достижения и фотографии выпускника)
4.	Информация о выпускнике(поля):
1.1 Кнопка редактирования данного выпускника (открывает новую страницу с полями для редактирования)
1.1 Фамилия, имя, отчество
1.2 Дата выпуска
1.3 Группа
1.4. Достижения
1.5 Фотография выпускника
5.	Нижняя панель:
4.1 Кнопки перелистывания выпускников

Форма добавления выпускника (Администратор):
Выбор необходимой группы или добавление новой группы (список предлагаемых групп):
•	Ввод ФИО 
•	Ввод года выпуска 
•	Ввод достижении выпускника
•	Выбор фотографии выпускника из галереи.
Кнопки добавления нового выпускника и выбора фотографии выпускника из галереи.





Форма подачи запроса на добавление нового выпускника (Клиент):
Выбор необходимой группы или добавление новой группы (список предлагаемых групп):
•	Ввод ФИО 
•	Ввод года выпуска 
•	Ввод достижении выпускника
•	Выбор фотографии выпускника из галереи.
Кнопки отправления запроса нового выпускника и выбора фотографии выпускника из галереи.
